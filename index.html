Moment 4 - JWT Auth

Registrera Logga in Dashboard

Den här webbplatsen demonstrerar autentisering med JWT (JSON Web Token). Du kan registrera ett konto, logga in och få åtkomst till en skyddad dashboard med data. JWT-token sparas i localStorage och används vid varje begäran till skyddade resurser.

Använd navigeringsmenyn ovan för att komma igång. Du måste logga in för att se Dashboard-sidan.
